# Dying Light Tweaks Manager
A simple Dying Light Tweaks Manager that allows you to:
- Specify arguments you want to use with your application
- Change video settings (even that ones that are not present in-game)
- Clear your logs, screenshots, crash dumps
- More support coming soon

# Installation
You can install this app 2 ways (only supports Windows):
- Download from releases
- It is available on scoop:
```
scoop bucket add dltweaks https://github.com/shwwwa/dl-bucket.git
scoop install dltweaks
```
